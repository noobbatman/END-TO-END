# CV Bullet Points

- Built a production-style document intelligence platform in Python with FastAPI, Celery, Redis, PostgreSQL, and Docker Compose to process scanned PDFs and images end to end.
- Designed a modular AI pipeline for OCR, document classification, key-value extraction, table parsing, entity extraction, confidence scoring, and audited JSON export across noisy real-world document layouts.
- Implemented human-in-the-loop review workflows for low-confidence fields, including reviewer correction APIs, audit trails, and correction persistence for future model-improvement analysis.
- Engineered extensible OCR and extraction interfaces with support for multiple document types, enabling invoice and bank-statement MVP delivery while scaffolding KYC, insurance, lease, and medical document expansion.
- Added production-oriented engineering features including structured logging, retryable background jobs, metrics instrumentation, automated tests, synthetic benchmark generation, and local one-command deployment with Docker Compose.
